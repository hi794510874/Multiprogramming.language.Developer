# 如何快速学会另一门编程语言？
掌握某一编程语言的最少必要知识，然后大量的去用。

掌握最少必要知识的方法论：
#### Step 1：精通一门编程语言，总结出最佳实践，并写成相关demo，这个demo要能让初学者7天内从10分达到85分以上。
#### Step 2：学新语言时就按这个demo再用新语言实现一遍，也许新语言更麻烦，也许更简单。但只要这么去做了，就能很快达到60分。
#### Step 3：真实使用后有新的发现及时更新，从而做到在极短的时间内再熟悉另一门语言。
